## Условие
Найти последнюю цифру n-ого числа Фибоначи,  
причем n может быть порядка миллиона.
## Решение
```
def fib(n):
    f1, f2 = 0, 1
    for i in range(n):
        f1, f2 = f2, (f1 + f2) % 10
    return f1


def main():
    n = int(input())
    print(fib(n))


if __name__ == "__main__":
    main()
```
## Примечание
Если а и b - это последние цифры чисел Fi и Fi+1, то (a+b)%10 - последняя цифра Fi+2 числа.
